  <summary><h2>About the book</summary>
<a href="https://www.packtpub.com/en-us/product/building-agent-powered-applications-9781807605162">
<img src="https://content.packt.com/B38358/cover_image_small.jpg" alt="Building Agent-Powered Applications, First Edition" height="256px" align="right">
</a>

Large language models can produce impressive demos, but turning them into dependable products takes more than better prompts. You need to understand model behavior, know when to use retrieval or fine-tuning, structure agents correctly, and evaluate systems before deployment.
Building Agent-Powered Applications gives you an end-to-end engineering perspective for creating production-ready generative AI solutions. Written by Microsoft Principal AI Engineer Vasyl Zvarydchuk, it helps software engineers, data scientists, and applied AI practitioners move from concepts to implementation. You’ll begin with AI, NLP, embeddings, transformers, and LLM behavior, then progress into prompt engineering, summarization, classification, extraction, reasoning, RAG, and fine-tuning.
The book also shows you how to design agentic workflows with tools, memory, planning, orchestration, and human-in-the-loop controls. You’ll learn how to evaluate quality with offline and online testing, task-specific metrics, LLM-as-a-judge methods, and responsible AI checks. Rather than treating prompting, RAG, fine-tuning, and agents as separate subjects, this book shows how they work together in real systems. By the end, you’ll be able to make better architectural trade-offs, reduce failure modes, and build scalable, trustworthy AI applications.
</details>
